## react-playground

### demo-react-site

start development webserver.

~~~console
$ yarn react:start
~~~

- view site in the browser: http://localhost:3030/

### demo-gatsby-site

start development webserver.

~~~console
$ yarn gatsby:start
~~~

- view site in the browser: http://localhost:8000/
- explore site's data and schema: http://localhost:8000/___graphql
